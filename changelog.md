**Description**: Design and implementation of "Leaf & Barrel", a premium cigar discovery and journaling web app. Includes Home, Database, and Cigar Detail views with a dark, minimal, Jony Ive-inspired aesthetic.
**Files**:
- index.html: Main entry point, layout shell. Refs: TailwindCSS, Google Fonts (Playfair Display, Inter), Lucide, GSAP.
- style.css: Custom typography, transitions, and scrollbar styling.
- app.js: Core logic, routing between views (Home, Database, Detail).
- components.js: Reusable UI patterns (Cigar Cards, Hero, Tags).
- data.js: Mock database of cigars and pairing recommendations.
